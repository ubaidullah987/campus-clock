# CampusClock Enterprise

An advanced facial recognition and geofencing-based attendance system using FaceAPI.js, Supabase, and Leaflet.

## Features
- **Biometric Check-In:** AI-powered facial recognition for student authentication.
- **Geofenced Campus:** GPS-locked attendance ensuring students are physically present.
- **Admin Console:** Real-time roster, dashboard analytics, and registration portal.
- **WhatsApp Integration:** Automated shortage alerts.
